# Software Contract Clause Checklist

## Licensing Terms

### Green Flags (Good)
- [ ] Clear definition of "user" or "seat"
- [ ] Reasonable overage provisions
- [ ] Right to reassign licenses
- [ ] Clear permitted use cases

### Red Flags (Concern)
- [ ] Audit rights with < 30 days notice
- [ ] Unlimited audit scope
- [ ] "All use" indemnification for audits
- [ ] Vague "reasonable use" without definition
- [ ] Automatic license reduction prohibited

---

## Data Rights

### Green Flags (Good)
- [ ] Customer owns all customer data
- [ ] Data portability in standard formats
- [ ] Data return/deletion upon termination
- [ ] Clear data processing locations
- [ ] Subprocessor notification required

### Red Flags (Concern)
- [ ] Vendor claims ownership of customer data
- [ ] "Anonymized" data usage without limits
- [ ] No data portability rights
- [ ] Unlimited data retention post-termination
- [ ] Broad data usage for "product improvement"
- [ ] Cross-customer data aggregation rights

---

## Liability

### Green Flags (Good)
- [ ] Liability cap = 12+ months fees
- [ ] Carve-outs for data breach
- [ ] Carve-outs for IP indemnity
- [ ] Mutual indemnification

### Red Flags (Concern)
- [ ] Liability cap < 6 months fees
- [ ] No carve-outs from cap
- [ ] One-way indemnification (customer only)
- [ ] "As-is" with no warranties
- [ ] Consequential damage waiver includes data loss
- [ ] Indemnification for "any claim" related to use

---

## Termination & Exit

### Green Flags (Good)
- [ ] 30+ day data retrieval period
- [ ] Data return in standard format
- [ ] Pro-rated refund for prepaid fees
- [ ] Termination for convenience allowed
- [ ] Wind-down assistance included

### Red Flags (Concern)
- [ ] No data return rights
- [ ] < 7 days to retrieve data
- [ ] Proprietary export format only
- [ ] No refund on early termination
- [ ] Termination triggers immediate data deletion
- [ ] "Survival" of payment for full term

---

## Auto-Renewal

### Green Flags (Good)
- [ ] Written renewal notice required
- [ ] 60+ days to opt-out
- [ ] Price increase cap (e.g., < 5%)
- [ ] Right to downgrade at renewal

### Red Flags (Concern)
- [ ] Auto-renewal without notice
- [ ] < 30 days opt-out window
- [ ] Unlimited price increases
- [ ] Multi-year auto-renewal
- [ ] Renewal at "then-current" rates

---

## Security & Compliance

### Green Flags (Good)
- [ ] SOC 2 Type II certified
- [ ] Breach notification within 72 hours
- [ ] Annual penetration testing
- [ ] Data encryption at rest and in transit
- [ ] GDPR/CCPA compliance commitment

### Red Flags (Concern)
- [ ] No security certifications
- [ ] No breach notification obligation
- [ ] Customer bears all compliance burden
- [ ] No security audit rights
- [ ] Vague "commercially reasonable security"

---

## SLA & Support

### Green Flags (Good)
- [ ] 99.9%+ uptime guarantee
- [ ] Service credits for downtime
- [ ] Defined severity levels
- [ ] Response time commitments
- [ ] Scheduled maintenance windows

### Red Flags (Concern)
- [ ] No uptime commitment
- [ ] Credits capped at < 1 month fees
- [ ] Excessive exclusions from uptime calc
- [ ] "Best efforts" support only
- [ ] No escalation path defined

---

## Negotiation Leverage Points

1. **Multi-year commitment** = Ask for price protection, better SLA
2. **Large deal size** = Push back on liability caps
3. **Regulated industry** = Require compliance addenda
4. **Data sensitivity** = Demand audit rights, breach notification
5. **Competitive alternatives** = More negotiating power overall
